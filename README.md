# Tobe Energy Data Room Dev

Development repo for data room components and investor materials.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000 to see the pipeline visualization.

## Components

- **PipelineNetwork** - Interactive pipeline visualization with Framer Motion animations

## Dev Workflow

- **Mac Studio (Ren)** pushes changes
- **MacBook (Colby)** pulls and reviews
- Once ready, components move to main tobe-website repo
